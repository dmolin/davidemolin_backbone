About me

I’m a Front End Developer, with a passion for Web Standards, Javascript programming, Front End Development in general and the World Wide Web.

I've been writing code professionally since 1995, starting with C and C++, then Java Web development (in 2001), eventually falling in love with Front End development (since 2005). I love technology, frameworks, design patterns and man-machine interactions. I'm always up for learning new stuff.

My current goal is to focus my efforts on Front End development (Javacript and dynamic languages) and Web Standards (HTML4-5+, CSS-2-3+).
